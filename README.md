# ProyectoParteIII-BLECH
Análisis dataset de jugadores del videojuego FIFA 2023 - evaluar el desempeño y potencial futbolístico de las distintas selecciones nacionales, especialmente Argentina, actual campeona del mundo. // Selección de features, entrenamiento de un clasificador que identifique jugadores "élite" (Overall ≥ 85), evaluación con métricas y conclusiones.
